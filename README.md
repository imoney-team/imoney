===========================================================================

`iMoney.js`(5.13kb min+gzip) is a minimalist JavaScript library for modern browsers  with a largely jQuery-compatible API.

## Install

```
npm install imoney --save
```

## Usage


```html
<div id="result"></div>
```

```js
import $ from "imoney";

var $result = $("#result");
$result.attr({
    title:"This is a title"
});
$result.html("<p>Hello World</p>") ;
```
