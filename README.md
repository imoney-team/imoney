===========================================================================

`iMoney.js` is a minimalist JavaScript library for modern browsers  with a largely jQuery-compatible API.

## Browser support

IE9+

## Install

```
npm install imoney
```

## Usage


```html
<div id="result"></div>
```

```js
var $ = require("imoney");

var $result = $("#result");
$result.attr({
    title:"This is a title"
});
$result.html("<p>Hello World</p>") ;
```
